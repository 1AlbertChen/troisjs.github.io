# Postprocessing

You can use EffectComposer (source) to easily add post processing effects.

Passes :

  • BokehPass (source)
  • FilmPass (source)
  • FXAAPass (source)
  • HalftonePass (source)
  • Renderpass (source)
  • SMAAPass (source)
  • UnrealBloomPass (source)

# Example

<template>
  <Renderer>
    <Camera :position="{ z: 10 }" />
    <Scene>
      <PointLight :position="{ z: 50 }" />
      <Box :size="1">
        <LambertMaterial />
      </Box>
    </Scene>
    <EffectComposer>
      <RenderPass />
      <UnrealBloomPass :strength="1" />
      <HalftonePass :radius="1" :scatter="0" />
    </EffectComposer>
  </Renderer>
</template>
