# Usage

TroisJS needs a Renderer, a Camera and a Scene :

<template>
  <Renderer>
    <Camera :position="{ z: 100 }" />
    <Scene>     
      <Box :size="10">
        <BasicMaterial />
      </Box>
    </Scene>
  </Renderer>
</template>
