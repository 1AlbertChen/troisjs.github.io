# Usage

TroisJS needs a Renderer, a Camera and a Scene, e.g. :

<template>
  <Renderer>
    <Camera :position="{ z: 100 }" />
    <Scene>     
      <Box :size="10">
        <BasicMaterial />
      </Box>
    </Scene>
  </Renderer>
</template>

# Renderer

See Renderer.js and THREE.WebGLRenderer.

# Props from THREE.WebGLRenderer
